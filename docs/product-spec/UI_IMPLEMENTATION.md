# UI 实现规范

所有图片位于 `assets/ui/`。禁止将图片直接设置为页面背景；必须用 WPF 控件、Skia 画布和图表重建。

## 共享壳

- 左侧导航宽度约 210px；
- 顶部信息栏约 92px；
- 内容卡圆角 6-10px；
- 页面背景与卡片背景分层；
- 统一 8px 间距系统；
- 主操作蓝色，危险/结束操作红色；
- 录制状态在右上始终可见。

## 01 实时采集

参考：`../assets/ui/01_realtime_capture.png`

- 左侧 60%-65% 为赛道画布；
- 右侧为速度、挡位、RPM、踏板、方向、圈速、G、轮胎、燃油；
- 底部显示圈与赛段时间轴；
- 开始记录禁用，结束记录高亮；
- 赛道上蓝色实际线、红绿目标线、白色赛车。
