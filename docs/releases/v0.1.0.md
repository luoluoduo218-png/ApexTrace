# ApexTrace v0.1.0 — First Public Release

ApexTrace is now available as an open-source, self-contained Windows x64 application for Le Mans Ultimate telemetry capture and analysis.

## Highlights

- Read-only live capture through the official `LMU_Data` shared-memory interface.
- Offline import of LMU native telemetry DuckDB recordings.
- Live telemetry instruments and measured racing-line reconstruction.
- Multi-lap overview, synchronized lap replay, and two-lap comparison.
- Automatic corner analysis and evidence-based setup A/B suggestions.
- Recovery of interrupted recordings from local checkpoints.
- Portable `.apextrace` session export and import.
- Chinese, English, and German language selections with external language-pack support.

## Installation

Download `ApexTrace-v0.1.0-win-x64.zip`, extract it to a writable folder, and run `ApexTrace.App.exe`. The package includes the required .NET runtime.

This first release is unsigned and may trigger a Windows SmartScreen prompt. It has been exercised against real LMU telemetry, but long endurance sessions, clean-machine installation, and the full DPI matrix still need broader validation.

Please use GitHub Issues for reproducible bugs and compatibility reports. Do not upload private telemetry, copyrighted game assets, protected archives, or credentials.
